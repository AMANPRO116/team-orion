Introduction

UAVs are critical in modern defense systems. Key requirements include low observability, high efficiency, and endurance. This project develops a stealth UAV inspired by fighter jet aerodynamics.

Design Inspiration

  • F-35 Lightning II inspiration
  • Blended Wing Body
  • Edge alignment
  • Reduced radar signature

Objective

  • Stealth UAV design
  • Subsonic operation
  • Surveillance & cargo missions

Theoretical Calculations

Lift: 220.5 N

Drag: 13.78 N

L/D Ratio: 16

Re ≈ 1.02×10⁶

System Architecture

  • Dual EDF propulsion
  • IMU based control
  • Communication module

Safety System

  • Thrust monitoring
  • Tilt validation
  • Communication check
