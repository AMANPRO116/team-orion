Introduction

Objective: Design a stealth UAV inspired by modern fighter aircraft with enhanced aerodynamic performance and mission adaptability.


The F-35D UAV project focuses on integrating aerodynamics, control systems, and structural design to create a high-efficiency unmanned platform.


  • Low observability using stealth shaping
  • High aerodynamic efficiency for extended endurance
  • Autonomous and semi-autonomous operation
  • Optimized for surveillance and reconnaissance missions

Design Inspiration

The design draws inspiration from 5th generation fighter aircraft to achieve stealth and performance.


  • Edge alignment to reduce radar cross-section
  • Blended wing-body configuration
  • Minimized vertical stabilizers
  • Smooth aerodynamic contours

System Architecture

  • Dual EDF (Electric Ducted Fan) propulsion system
  • Flight controller with IMU-based stabilization
  • Sensor integration for environmental awareness
  • Communication module for remote control
  • Power distribution system for efficient energy usage

Safety System (CAASS)

Control Authority Aware Safety System (CAASS) ensures stable and safe operation by continuously monitoring system parameters.


  • Thrust effectiveness monitoring
  • Tilt response validation
  • IMU-based motion tracking
  • Power system health monitoring
  • Communication integrity checks
  • Prediction of unstable flight conditions

Emergency Detachment System

The aircraft is equipped with a detachable front module designed for critical failure scenarios.


  • Front EDF module detaches during rear propulsion failure
  • Ensures survival of critical onboard data
  • Improves mission success in hostile environments
  • Prevents total system loss during emergencies

Theoretical Calculations

Lift: 220.5 N

Drag: 13.78 N

L/D Ratio: 16


These values indicate high aerodynamic efficiency suitable for long-duration missions.
