Theoretical Calculations based on simulations

Reynolds Number

Re = ρVL / μ

Re = (1.225 × 30 × 0.5) / (1.8 × 10^{-5})

Final Answer: Re ≈ 1.02 × 10^6

Lift Calculation

L = ½ ρ V² S C_L

L = ½ × 1.225 × (30)² × 0.5 × 0.8

Final Answer: L = 220.5 N

Drag Calculation

D = ½ ρ V² S C_D

D = ½ × 1.225 × (30)² × 0.5 × 0.05

Final Answer: D = 13.78 N

Aerodynamic Efficiency

L/D = C_L / C_D = 0.8 / 0.05

Final Answer: L/D = 16

Wing profile

The NACA 23012 is a 5-digit airfoil from the National Advisory Committee for Aeronautics (NACA) series, designed for good lift and stable performance at moderate speeds.

The 5-digit NACA system encodes aerodynamic properties:

  • 2 → Design lift coefficient ≈ 0.3
  • 3 → Position of maximum camber (≈ 15% of chord)
  • 0 → Type of camber (standard camber line)
  • 12 → Maximum thickness = 12% of chord

Moderate camber → Produces lift even at low angles of attack

Thickness (12%) → Good structural strength + space for internal components

Low pitching moment → Stable and predictable behavior

Good stall characteristics → Smooth, gradual stall

Emergency Detaching System

The UAV incorporates an emergency detachment system in which the front section—housing the forward EDF unit and critical onboard systems—can be separated during high-risk scenarios. This mechanism is designed to activate in situations such as rear propulsion (EDF) failure or imminent external threats (e.g., incoming missile detection).

The primary objective is to preserve mission-critical data and essential avionics by isolating the front module from the main airframe. Upon detachment, the front section is capable of independent stabilization and controlled descent/recovery, ensuring that valuable reconnaissance information gathered during close-range surveillance of hostile zones is not lost.

The system is engineered for rapid response, structural reliability, and minimal aerodynamic penalty during normal operation.

SYSTEM MODES

Dual-Mode Architecture

The system operates in two configurations:

  1. Combined Mode
    Functions as a hybrid VTOL aircraft
    Distributed propulsion
    Centralized control
  2. Separated Mode
    Splits into two independent UAVs
    Each module stabilizes autonomously
    Optional cooperative operation

ELECTRICAL SYSTEM ARCHITECTURE

Distributed Electrical Design

Two independent power domains:

  • Front Module → FC1 + Battery 1
  • Rear Module → FC2 + Battery 2

Each module operates:

  • Independently (after separation)
  • As a unified system (when docked)

POWER & PROPULSION SYSTEM

Power System

Dedicated batteries for each module

Separate regulated rails: 6-7.4V → Servos 5V → Controllers & communication

Propulsion Configuration

Front EDF (tilting) Rear EDFs (2×, tilting) Independent ESC control (no mechanical linkage)

AVIONICS & COMMUNICATION

Flight Controllers

  • FC2 (Rear) → Master Controller
  • FC1 (Front) → Slave / Independent Controller

Communication

  • Wired link (UART/CAN) when docked
  • Wireless link (LoRa/RF) when separated

Data Exchange

  • Thrust commands
  • Tilt angles (front & rear)
  • Flight mode
  • Status feedback

Key Insight: Communication replaces mechanical coupling between propulsion units.

CONTROL SYSTEM ARCHITECTURE

Control Philosophy: Distributed hardware with centralized control authority in combined mode.

Combined Mode (Docked Operation)

FC2 (Rear) → Master

Computes:

  • Total lift
  • Pitch & roll stability
  • Transition control

Rear EDFs

Tilt range: 90° → 0°

Role:

  • 90° → Full vertical lift (hover)
  • Gradually reduce tilt → shift to forward thrust

Thrust Vectoring Logic (CRITICAL)

Front EDF — Tilt range: 0° → 30°

Role:

  • 0° → Hover assist
  • 10°-30° → Transition + forward thrust

Control Execution

FC1 executes: Ailerons, Elevator, Front EDF tilt

Separated Mode

FC1 → Independent fixed-wing UAV
FC2 → Independent hover UAV

SAFETY SYSTEM (CAASS + RED FLAGS)

CAASS (CONTROL AUTHORITY AWARE SAFETY SYSTEM)

  • Monitors thrust effectiveness
  • Checks tilt response (front & rear)
  • Tracks aircraft motion (IMU)
  • Observes power limits
  • Monitors communication health

RED FLAGS

  • High-thrust descent
  • Tilt mismatch
  • Power overload / voltage drop
  • Control delay
  • Communication failure
  • Predicted instability

DECISION LOGIC

Requires multiple condition confirmation

Actions

  • Abort transition
  • Return to hover
  • Enter safe mode

FAILSAFE & SYSTEM INTEGRATION

ELECTRICAL SAFETY

  • Dual battery system
  • Isolated power lines

CONTROL SAFETY

  • Master-slave architecture
  • Continuous communication check

FAILURE HANDLING

  • Front failure → controlled descent
  • Rear thrust issue → abort transition
  • Comm loss → safe mode

FINAL CAPABILITY

  • Smooth hover-to-cruise transition
  • Unified aircraft operation
  • Splits into two independent UAVs
